# Directory Structure:

Identical data was retrieved from [Marcion](http://marcion.sourceforge.net/) in both CSV and HTML formats. The directory contains the raw data, processing scripts, as well as some utilities.

## csv

This contains the data in CSV format, and the code for processing this format.

## html

This contains the data in HTML format, and the code for processing this format.

## utils

This contains some utility scripts.
